This is a heading in Open Sans using CSS3

This is a paragraph in Open Sans using CSS3

This font is designed for display on a computer. This font was comissioned by Google for free use. According to Wikipedia: it was developed with an "upright stress, open forms and a neutral, yet friendly appearance" and is "optimized for legibility across print, web, and mobile interfaces."

@font-face { font-family: 'Open Sans'; font-style: normal; font-weight: 400; src: local('Open Sans'), local('OpenSans'), url('fonts/opensans.woff') format('woff'); } p { font-family: 'Open Sans', Arial, sans-serif; }
